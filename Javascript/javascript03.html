01. 함수 : 선언적 함수

선언적 함수(Declarative Function)는 이름을 가진 함수를 정의하는 방법 중 하나입니다.
이 방식은 함수를 선언하고 이름을 지정함으로써 언제든지 해당 이름을 사용하여 함수를 호출할 수 있게 합니다. 

{
        function func() {
            console.log("01. 함수가 실행되었습니다.");
        };
        func();
}
결과 확인하기
01. 함수가 실행되었습니다.

02. 함수 : 익명 함수

익명함수는 이름이 없는 함수로, 주로 일회성 함수나 콜백으로 사용됩니다.
이 함수는 변수에 할당될 수 있으며, 할당된 변수를 통해 호출됩니다. 

{
        const func = function () {
            console.log("01. 함수가 실행되었습니다.");
        };
        func();
}
결과 확인하기
02. 함수가 실행되었습니다.

03. 함수 : 매개변수 함수

매개변수를 가진 함수는 입력 값을 받아들여 함수 내부에서 사용할 수 있도록 설계된 함수입니다. 이 매개변수들은 함수가 호출될 때 제공되며, 함수 내에서는 변수처럼 사용됩니다. 

{
        // 선언적 함수
        function func(str) {
            console.log(str);
        };
        func("03. 함수가 실행되었습니다.");

        // 익명 함수
        const func1 = function (str) {
            console.log(str);
        };
        func1("03-1. 함수가 실행되었습니다.")
}
결과 확인하기
03. 함수가 실행되었습니다. // 선언적 함수 03-1. 함수가 실행되었습니다. // 익명 함수

04. 함수 : 리턴값 함수

리턴값을 가진 함수는 함수의 실행 결과를 호출한 곳으로 반환하는 함수입니다. 함수 내에서 return 키워드 뒤에 오는 값을 호출한 위치로 돌려보내며, 이 리턴값을 이용해 함수 외부에서 다양한 연산이나 처리를 할 수 있습니다. 

{
        // 선언적 함수
        function func() {
            return "04. 함수가 실행되었습니다."
        }
        console.log(func());

        // 익명 함수
        const func1 = function (str) {
            return "04-1. 함수가 실행되었습니다."
        };
        console.log(func1());
}
결과 확인하기
04. 함수가 실행되었습니다. // 선언적 함수 04-1. 함수가 실행되었습니다. // 익명 함수

05. 함수 : 매개변수 + 리턴값 함수

매개변수와 리턴값을 모두 가진 함수는 외부로부터 입력 값을 받아들이고, 처리 후 그 결과를 반환하는 함수입니다.
이러한 함수는 입력(매개변수)에 대해 정의된 연산이나 처리를 수행한 뒤, 그 결과(리턴값)를 호출한 측에 돌려주는 구조로 되어 있습니다. 

{
        // 선언적 함수
        function func(str) {
            return str;
        }
        console.log(func("05. 함수가 실행되었습니다."));

        // 익명함수
        const func1 = function (str) {
            return str;
        }
        console.log(func1("05-1. 함수가 실행되었습니다."));
}
결과 확인하기
05. 함수가 실행되었습니다. // 선언적 함수 05-1. 함수가 실행되었습니다. // 익명 함수

10. 화살표 함수 : 매개변수 + 리턴값

리턴값 함수의 약식입니다. 

{
        // 익명 함수(화살표) + 매개변수 + 리턴값
        const func = (str) => {
            return str;
        };
        console.log(func("10. 함수가 실행되었습니다."));


        // 매개변수가 하나이면 괄호 생략 가능
        const func2 = str => {
            return str;
        }
        console.log(func2("10-1. 함수가 실행되었습니다."));

        // 리턴 생략
        const func3 = str => str;
        console.log(func3("10-2. 함수가 실행되었습니다."));

        // 선언적 함수(가독성X)
        func4 = str => str;
        console.log(func4("10-3. 함수가 실행되었습니다."));
}
결과 확인하기
10. 함수가 실행되었습니다. // 익명 함수(화살표) + 매개변수 + 리턴값 10-1. 함수가 실행되었습니다. // 매개변수가 하나이면 괄호 생략 가능 10-2. 함수가 실행되었습니다. // 리턴 생략 10-3. 함수가 실행되었습니다. // 선언적 함수(가독성X)
