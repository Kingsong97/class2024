01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 지정하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(y);
        console.log(z);
}

02. 변수 : 데이터 저장 + 데이터 변경

JavaScript에서 변수를 정의하고 이후에 변수들의 값을 언제든지 변경할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x = 200;
        y = 300;
        z = "jquery";

        console.log(x)

코드의 결과로 콘솔에는 변수 x, y, z의 최종값이 출력됩니다.

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열 구조 분해 할당

배열 구조 분해 할당은 배열의 요소를 추출하여 변수에 할당하는 방식을 의미합니다.
이것은 배열을 선언할 때 괄호 [ ] 안에 있는 요소를 변수로 분해하여 할당하는 방법입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);
}

아래의 코드에서는 const [a, b, c] = arr;는 배열 arr의 구조를 분해하여 각 요소를 변수 a, b, c에 할당하는 것을 의미합니다.

12. 배열 : 데이터 저장(여러개) : 배열 펼침 연산자

배열 펼침 연산자는 ... 기호를 사용하여 배열을 개별 요소로 분해하거나 여러 배열을 결합할 때 사용하는 JavaScript의 문법입니다.
이 연산자는 배열의 요소를 펼쳐서 각각의 개별적인 값으로 사용할 수 있게 합니다. 

{
    const arr1 = [100, 200];
    const arr2 = [300, 400];
    const merge = [...arr1, ...arr2];
    const merge1 = [...arr1, "javascript", ...arr2, "jquery"];

    console.log(arr1);
    console.log(...arr1);
    console.log(...arr1, ...arr2);
    console.log(merge);
    console.log(merge1);
    console.log(arr1.concat(arr2));
}

위의 코드에서는첫 번째 console.log(...arr1);에서 배열 arr1의 요소들이 개별적인 값으로 펼쳐져서 출력됩니다.
두 번째 console.log(...arr1, ...arr2);에서는 두 개의 배열 arr1과 arr2의 요소들이 개별적인 값으로 펼쳐져서 함께 출력됩니다.

[100, 200, 300, 400]

20. 객체 : 데이터 저장(키와값) : 표현방법 8 : 객체 + 함수가 있는 방식

        obj.f();

22. 객체 : 데이터 저장(키와값) : 표현방법 10 : 객체 펼침 연산자

        console.log(obj2.a);
        console.log(obj2.b);
        console.log(obj2.c);
        console.log(obj2.d);
        console.log(obj3);
}

23. 연산자

프로그래밍에서 값을 조작하고 계산하는 데 사용되는 기호 또는 기능입니다.
다양한 유형의 연산자가 있으며, 각각의 연산자는 특정한 작업을 수행합니다.

1. 산술 연산자 (Arithmetic Operators)

- 산술 연산자는 숫자 값을 사용하여 계산을 수행합니다.
• + : 덧셈
• - : 뺄셈
• * : 곱셈
• / : 나눗셈
• % : 나머지

[증가/감소 연산자]
• ++ : 증가 연산자. 변수의 값을 1씩 증가
• -- : 감소 연산자. 변수의 값을 1씩 감소 

{
    let a = 10;
    let b = 5;
    
    console.log(a + b); //결과: 15
    console.log(a - b); //결과: 5
    console.log(a * b); //결과: 50
    console.log(a / b); //결과: 2
    console.log(a % b); //결과: 0
    
    [증가/감소 연산자] ============================

    let x = 5;
    console.log(++x); // 출력: 6 (x가 증가된 후에 출력)
    console.log(x++); // 출력: 6 (x가 출력된 후에 증가)
    console.log(x);   // 출력: 7 (x의 값은 이전에 증가되어 6에서 7로 변함)

    let y = 10;

    console.log(--y); // 출력: 9 (y가 감소된 후에 출력)
    console.log(y--); // 출력: 9 (y가 출력된 후에 감소)
    console.log(y);   // 출력: 8 (y의 값은 이전에 감소되어 9에서 8로 변함)
}
2. 할당 연산자 (Assignment Operators)

- 할당 연산자는 변수에 값을 할당합니다.
• = : 변수에 값을 할당합니다.
• += : 현재 변수 값과 더하여 변수에 할당합니다.
• -= : 현재 변수 값에서 빼어 변수에 할당합니다.
• *= : 현재 변수 값과 곱하여 변수에 할당합니다.
• /= : 현재 변수 값을 나누어 변수에 할당합니다.
• %= : 현재 변수 값을 나눈 후 나머지를 변수에 할당합니다.

{
    let x = 10;
    x += 5; // x = x + 5;
    console.log(x); // 출력 결과: 15
    
    let y = 20;
    y -= 3; // y = y - 3;
    console.log(y); // 출력 결과: 17
    
    let z = 5;
    z *= 2; // z = z * 2;
    console.log(z); // 출력 결과: 10
}
3. 논리 연산자 (Logical Operators)

- 논리 연산자는 논리적 표현식을 평가하고 불리언 값을 반환합니다.
• && : AND 연산자
• || : OR 연산자
• ! : NOT 연산자

{
    let x = 10;
    let y = 5;
    
    console.log(x > 5 && y < 10); // 결과: true
    console.log(x > 5 || y > 10); // 결과: true
    console.log(!(x == 5)); // 결과: true
}
4. 비교 연산자 (Comparison Operators)

- 두 개의 값을 비교하고, 그 결과를 불리언 값으로 반환합니다. 주로 조건문이나 반복문에서 사용됩니다.
• == : 동등 연산자 (값 비교)
• === : 엄격한 동등 연산자 (값, Type 비교)
• != : 부등 연산자
• !== : 엄격한 부등 연산자 (값, Type 비교)

[관계연산자]
<: 작은지를 비교합니다.
>: 큰지를 비교합니다.
<=: 작거나 같은지를 비교합니다.
>=: 크거나 같은지를 비교합니다.

{
    let a = 5; // 변수 a에 5 할당
    let b = "5"; // 변수 b에 문자열 "5" 할당

    console.log(a == b); // true (형 변환 후 값이 같으므로)
    console.log(a === b); // false (타입이 다르므로)
    console.log(a != b); // false (형 변환 후 값이 같으므로)
    console.log(a !== b); // true (타입이 다르므로)
    console.log(a > b); // false (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
    console.log(a < b); // false (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
    console.log(a >= b); // true (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
    console.log(a <= b); // true (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
}
5. 삼항 연산자(Ternary Operator)

- 조건식을 평가하고 그 결과에 따라 두 가지 표현식 중 하나를 반환합니다.
[형식: condition ? expression1 : expression]
- condition : 평가할 조건식
- expression1, expression2 : 조건식이 참(true)일 때와 거짓(false)일 때 각각 반환될 표현식

{
    let num = 10;
    let result = (num % 2 === 0) ? "짝수" : "홀수";
    console.log(result); // 출력: "짝수" (10은 짝수이므로)
}
