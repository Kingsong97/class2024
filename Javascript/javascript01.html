01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 지정하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(x);
        console.log(y);
        console.log(z);
}

02. 변수 : 데이터 저장 + 데이터 변경

JavaScript에서 변수를 정의하고 이후에 변수들의 값을 언제든지 변경할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x = 200;
        y = 300;
        z = "jquery";

        console.log(x)
        console.log(y)
        console.log(z)
}

let 키워드를 사용하여 변수를 선언하고, 변수에 값을 할당할 때 = 연산자를 사용합니다.
코드의 결과로 콘솔에는 변수 x, y, z의 최종값이 출력됩니다.

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

JavaScript에서 변수를 정의하고 이후에 변수들의 값에 증감 시킬 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x += 200;
        // x = x + 200
        y += 300;
        z += "jquery"

        console.log(x);
        console.log(y);
        console.log(z);
}

숫자열엔 증감시키는 증감 연산자를 사용하여 값을 변경합니다.
문자열엔 오직 증가 연산자만 사용하여 문자를 연결할 수 있습니다.

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언

배열 선언은 프로그래밍에서 배열을 생성하고 초기화하는 과정을 말합니다.
배열은 여러 값을 하나의 변수에 저장하는 자료 구조입니다. 

{
        const arr = new Array();
   
        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
}

빈 배열을 생성한 후에 각 요소에 값을 할당하고, 배열의 각 요소 값을 콘솔에 출력합니다.
배열은 숫자나 문자열 등 다양한 타입의 값을 저장할 수 있으며, 각 요소는 인덱스를 사용하여 접근할 수 있습니다.
console.log() 함수를 사용하여 배열의 특정 요소 값을 콘솔에 출력할 수 있습니다.

10. 배열 : 데이터 저장(여러개) : 표현방법 6 : 다차원 배열

2차 배열에서는 배열 안에 또 다른 배열이 포함됩니다.
2차 배열을 사용하면 하나의 배열 안에 여러 개의 배열을 넣어 복잡한 데이터 구조를 표현할 수 있습니다. 

{
        const arr = [100, [200, 300, [400, 500, [600]]], ["javascript", "Jquery"]];

        console.log(arr[0]);
        console.log(arr[1][0]);
        console.log(arr[1][1]);
        console.log(arr[2][0]);
        console.log(arr[2][1]);
        console.log(arr[1][2][2][0]);
}

위와 같이 2차 배열을 구현할 때는 각 행을 배열로 나타내고, 각 행을 포함하는 배열을 다시 하나의 배열에 넣어서 표현합니다.

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열 구조 분해 할당

배열 구조 분해 할당은 배열의 요소를 추출하여 변수에 할당하는 방식을 의미합니다.
이것은 배열을 선언할 때 괄호 [ ] 안에 있는 요소를 변수로 분해하여 할당하는 방법입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);
}

아래의 코드에서는 const [a, b, c] = arr;는 배열 arr의 구조를 분해하여 각 요소를 변수 a, b, c에 할당하는 것을 의미합니다.

12. 배열 : 데이터 저장(여러개) : 배열 펼침 연산자

배열 펼침 연산자는 ... 기호를 사용하여 배열을 개별 요소로 분해하거나 여러 배열을 결합할 때 사용하는 JavaScript의 문법입니다.
이 연산자는 배열의 요소를 펼쳐서 각각의 개별적인 값으로 사용할 수 있게 합니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "Jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);
        console.log(...arr1);
        console.log(...arr1, ...arr2); 
        console.log(...arr3); 
}

위의 코드에서는첫 번째 console.log(...arr1);에서 배열 arr1의 요소들이 개별적인 값으로 펼쳐져서 출력됩니다.
두 번째 console.log(...arr1, ...arr2);에서는 두 개의 배열 arr1과 arr2의 요소들이 개별적인 값으로 펼쳐져서 함께 출력됩니다.
