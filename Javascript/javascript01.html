01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 지정하고 참조하는데 사용합니다. 변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        console.log(x);
        console.log(y);
        console.log(z);
}

JavaScript에서 변수를 정의하고 이후에 변수들의 값을 언제든지 변경할 수 있습니다. 

{
        let x = 100;

{
        let x = 100;

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열 구조 분해 할당

배열 구조 분해 할당은 배열의 요소를 추출하여 변수에 할당하는 방식을 의미합니다.
이것은 배열을 선언할 때 괄호 [ ] 안에 있는 요소를 변수로 분해하여 할당하는 방법입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);
}

아래의 코드에서는 const [a, b, c] = arr;는 배열 arr의 구조를 분해하여 각 요소를 변수 a, b, c에 할당하는 것을 의미합니다.

12. 배열 : 데이터 저장(여러개) : 배열 펼침 연산자

배열 펼침 연산자는 ... 기호를 사용하여 배열을 개별 요소로 분해하거나 여러 배열을 결합할 때 사용하는 JavaScript의 문법입니다.
이 연산자는 배열의 요소를 펼쳐서 각각의 개별적인 값으로 사용할 수 있게 합니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "Jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);
        console.log(...arr1);
        console.log(...arr1, ...arr2); 
        console.log(...arr3); 
}

위의 코드에서는첫 번째 console.log(...arr1);에서 배열 arr1의 요소들이 개별적인 값으로 펼쳐져서 출력됩니다.
두 번째 console.log(...arr1, ...arr2);에서는 두 개의 배열 arr1과 arr2의 요소들이 개별적인 값으로 펼쳐져서 함께 출력됩니다.

21. 객체 : 데이터 저장(키와값) : 표현방법 9 : 객체구조 분해 할당

객체 분해 할당은 JavaScript에서 객체의 속성을 추출하여 해당 속성 값을 변수로 할당하는 방법입니다.
중괄호 {} 안에 추출하고자 하는 속성 이름을 명시하고, 이에 해당하는 값은 변수에 할당됩니다. 

{
        const obj = {
            a: 100,
            b: 200,
            c: "javascript",
        }

        const { a, b, c } = obj;

        console.log(a);
        console.log(b);
        console.log(c);
}
